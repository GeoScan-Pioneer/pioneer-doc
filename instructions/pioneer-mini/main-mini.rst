Геоскан "Пионер Мини"
=====================

.. image:: media/pioneermini.png
   :align: center
   :width: 420

.. sidebar:: Пионер Мини

	Образовательный квадрокоптер с функциями доверенной среды для первого знакомства с автономными летательными аппаратами.

	Пионер Мини - предназначен для комплексного обучения в рамках проведения уроков технологий или занятий по робототехнике в помещениях (Школах, ЦМИТов, Фаблабов, кружков, секций). 

	Квадрокоптер разработан в рамках концепции STEM-образования, позволяя ученикам развиваться сразу в нескольких предметных областях – программировании, физике, математике. Пионер Мини может летать в помещении полностью автономно по написанной учениками программе, открывая широкие возможности для проведения учебных занятий и соревнований по робототехнике.

.. sidebar:: Возможности Мини:

   *  Получить базовые навыки пилотирования;
   *  Ознакомиться с устройством и принципом работы квадрокоптера;
   *  Изучить на реальном примере физику полёта;
   *  Научиться программировать электронные устройства;
   *  Освоить новое интересное хобби.


.. sidebar:: Преимущества Мини:

   *  Полностью автономный полет по программе;
   *  Учебно-методическое пособие для учителей и учеников;
   *  Возможность управления с мобильных устройств;
   *  Возможность установки дополнительных модулей;
   *  Небольшой вес и размер, простая сборка;
   *  Поворотная фото/видео камера;
   *  Поддержка ультразвуковой (УЗ) и инфракрасной (ИК) систем навигации в помещении;
   *  Широкие возможности для программирования благодаря ESP-32;
   *  Возможность зарядки аккумулятора через micro-USB, не вынимая из коптера.

.. sidebar:: Функция доверенной среды:

   *  Фиксирует ключевые параметры полёта коптера в "чёрном ящике".
   *  Просмотр логов возможен с помощью приложения Pioneer Station.
   *  Обеспечивает контроль исполнения полетного задания. Реализуется возможность по установке ограничения на зону полёта в локальной системе навигации.


.. important:: Квадрокоптер предназначен для полёта в помещениях.

.. important:: Перед осуществлением полета, обязательно обновите прошивку и параметры автопилота (если версия автопилота ниже, чем 1.6.7747). Полет с устаревшей версией прошивки автопилота может быть небезопасен!
               Проверьте версию прошивки автопилота в Pioneer Station и при необходимости перейдите на страницу
               :doc:`settings/firmware_upgrade`.
