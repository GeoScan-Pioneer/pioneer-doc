Программирование на Python
==========================

.. important:: На данный момент библиотека **pioneer_sdk** работает только на квадрокоптере "Пионер Мини". В дальнейшем будет добавлена поддержка других устройств.

Библиотека **pioneer_sdk** (`ссылка на GitHub <https://github.com/geoscan/pioneer_sdk>`__) позволяет программировать "Пионер Мини" на языке Python. С примерами Python-скриптов для квадрокоптера можно ознакомиться в папке **examples** в репозитории на GitHub.
