Пилотирование
=============

.. attention:: Блок находится на обновлении.

.. note:: Данный блок был написан с учётом использования квадрокоптера "Геоскан Пионер", но он также подойдет для обучения базовым навыкам пилотирования на других квадрокоптерах.

Настоящий блок "Базы Знаний" содержит основную информацию о
двух основных режимах ручного управления квадрокоптером визуальном и
FPV. Изучив данный блок, вы сможете приобрести базовые навыки и знания, необходимые для первого
полета.
